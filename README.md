l objetivo de este proyecto es clasificar series de tiempo correspondientes a señales EEG de personas con TDAH y sujetos control, utilizando un enfoque bayesiano no paramétrico: Procesos Gaussianos para clasificación. Para ello, se representan las señales EEG sintéticas mediante Modelos Ocultos de Markov (HMM), capturando la dinámica de los estados latentes que describen los patrones temporales relevantes.
